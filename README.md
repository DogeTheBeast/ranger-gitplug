gitplug
=======
git plugin for ranger file manager.

#### Working command list:
* ```:git init```
* ```:git status```
* ```:git clone <url>```
* ```:git add <file>```
* ```:git rm <file>```
* ```:git restore <file>``` 
* ```:git commit <text>```
